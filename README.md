# 🚀 DevPlan Pro — Software Engineering Planning Suite

A full-stack browser-based software engineering planning tool built with React.js.

## 🔧 Features
- COCOMO I & II Cost Estimation
- SDLC Model Simulator (Waterfall, Agile, Spiral, V-Model, Incremental)
- Sprint Board with Kanban & Burndown Charts
- Risk Analysis Heatmap (P × I Matrix)
- Earned Value Management (SPI, CPI, EAC)
- Milestone Tracker with Animations
- Analytics Dashboard with Progress Rings
- Aurora WebGL Animated Background
- Dark/Light Theme + Accent Color Picker
- Global Search (Ctrl+K)

## 🛠️ Tech Stack
- React.js
- JavaScript ES6+
- WebGL2 / GLSL (Aurora animation)
- Canvas API (Charts)
- CSS3 Animations
