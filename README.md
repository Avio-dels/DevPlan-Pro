# 🚀 DevPlan Pro — Software Engineering Planning Suite

**DevPlan Pro** is a full-stack, browser-based software engineering planning tool designed to streamline project estimation, simulation, and tracking. It integrates industry-standard models like COCOMO, SDLC simulations, and Agile workflows into a single interactive platform.

## 🔧 Features

### 📊 Project Planning & Estimation

* COCOMO I & II Cost Estimation Models
* Earned Value Management (SPI, CPI, EAC)
* Milestone Tracking with interactive visuals

### 🔄 Development Simulation

* SDLC Model Simulator:

  * Waterfall
  * Agile
  * Spiral
  * V-Model
  * Incremental

### 📌 Agile Workflow Tools

* Sprint Board with Kanban system
* Burndown Charts for sprint tracking

### ⚠️ Risk & Analytics

* Risk Analysis Heatmap (Probability × Impact Matrix)
* Analytics Dashboard with real-time progress indicators

### 🎨 UI/UX Enhancements

* Aurora WebGL Animated Background
* Dark/Light Theme with Accent Color Picker
* Global Search (Ctrl + K)

## ▶️ Getting Started

### Prerequisites

* Node.js (v14 or higher)
* npm or yarn

### Installation

```bash
git clone https://github.com/Avio-dels/devplan-pro.git
cd devplan-pro
npm install
npm start
```
